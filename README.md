# Supercar Rental Project

- ReactJs
- Vite
- React Router DOM
- GSAP (GreenSock Animation Platform)
- Font Awesome
- React CountUp
- ESLint

### Link to project: https://supercar-rental.netlify.app/

### Getting Started.

Make sure you have Node.js and npm

    git clone https://github.com/ddimkou/supercar-rent.git

    cd supercar-rent

    npm install

    npm run dev
